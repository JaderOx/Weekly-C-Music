# C++Music

## 使用步骤

 第一种方法
	直接复制 `./Weekly-C++Music/music.cpp`里的代码到 `Visual Studio`空项目中，运行。

第二种方法

1. `fork`一个远程仓库
2. `clone`你的远程仓库到本地
3. 本地文件夹用 `Visual Studio`打开项目文件 `Weekly-C++Music.sln`

- 注:`VS`自带 `MSVC`环境，因此可以直接运行代码。而 `VSCode`需要手动配置 `MSVC`环境，否则无法调用 `MIDI`相关函数，无法正常启动。
